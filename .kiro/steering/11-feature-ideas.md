# CRF 機能アイデア一覧

**更新日**: 2026年3月24日

---

## 🔬 コア機能（MVP）

### 1. クリーチャー図鑑
- 一覧ページ・詳細ページ
- フィルター・ソート機能
- ステータス: 設計済み（Pencilデザイン完了）

### 2. 特徴検索（Trait Search）
- 「翼がある」「水棲」「夜行性」「体長3m以上」などの特徴タグで絞り込み
- 目撃情報から「こんな生物を見た」→ 該当候補を表示する逆引き検索
- 研究機関のデータベースとして自然なUX

### 3. 地域別ブラウズ
- 地図UIまたは地域カードから探索
- 日本、北米、ヨーロッパなど地域ごとのクリーチャー一覧

---

## 🏛️ 研究機関らしい機能

### 4. 調査依頼（Investigation Request）
- ユーザーが目撃情報や調査してほしい生物を投稿できるフォーム
- SCP財団の「報告書提出」的な雰囲気
- フォーム項目例: 目撃日時、場所、生物の特徴、状況説明、証拠写真

### 5. 危険度・信憑性レーティング
- 各クリーチャーにCRF独自の分類等級を付与（例: Class-I〜V）
- 信憑性スコア（目撃件数、物的証拠の有無などに基づく）
- 研究機関っぽさが一気に増す差別化要素

### 6. 調査ファイル風UI
- 詳細ページを「機密文書」風のレイアウトにする
- ファイル番号（例: CRF-0042）
- 分類コード
- 調査ステータス（調査中 / 確認済 / 未解決 / 機密）
- SCP財団的な世界観との相性◎

---

## 🎯 エンゲージメント系

### 7. 目撃報告マップ
- 世界地図上に目撃ポイントをプロット
- ユーザー投稿も可能にして参加型に
- Google Maps API or Mapbox

### 8. 週刊/月刊レポート（ブログ）
- 「CRF調査報告」として記事を発信
- SEO流入の柱になる
- コンテンツ例: 新規調査報告、地域特集、歴史考察

### 9. クリーチャー比較
- 2体を並べてスペック比較できる機能
- サイズ、危険度、目撃件数、生息地域などを比較表示

---

## 💰 収益化に繋がる機能

### 10. 関連グッズセクション
- 各クリーチャー詳細ページからEtsyショップへの導線
- 「このクリーチャーのグッズを見る」ボタン

### 11. フィールドガイド（PDF）
- 地域別のガイドをダウンロード販売
- またはメルマガ登録特典として無料配布
- リード獲得にも使える

---

## 🚀 将来的に面白そうなもの

### 12. クリーチャー診断クイズ
- 「あなたに似ているUMAは？」的なバイラルコンテンツ
- SNSシェアとの相性◎

### 13. コミュニティフォーラム
- 研究員（ユーザー）同士の議論の場
- 目撃情報の共有、考察スレッド

### 14. 匿名掲示板（コメント機能）
- 各詳細ページ下部に「調査員間情報共有チャンネル」風の掲示板
- 匿名投稿（ニックネーム任意）
- 実装タスク:
  - [ ] Supabase or DynamoDB でコメントテーブル
  - [ ] API Route (`/api/comments`) で CRUD
  - [ ] reCAPTCHA v3 でボット対策
  - [ ] NGワードフィルター
  - [ ] レート制限（連投防止）
