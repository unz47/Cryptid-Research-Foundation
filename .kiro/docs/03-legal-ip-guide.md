## 📖 著作権の基礎知識

### 著作権とは

**定義**: 創作的な表現を保護する権利

**保護対象**:
- ✅ イラスト、絵画、写真
- ✅ 文章、詩、小説
- ✅ 音楽、映像
- ✅ キャラクターデザイン

**保護されないもの**:
- ❌ アイデアそのもの（例: "ツチノコ"という概念）
- ❌ 事実や歴史的記述
- ❌ 手法やスタイル（例: "浮世絵風"）
- ❌ ありふれた表現

### 著作権の期間

| 国・地域 | 保護期間 | パブリックドメインになる年 |
|---------|---------|--------------------------|
| 日本 | 死後70年 | 1953年以前没の作者の作品 |
| アメリカ | 死後70年 | 1928年以前発行の作品 |
| EU | 死後70年 | 1953年以前没の作者の作品 |

**重要な例外**:
- 古代神話・民間伝承: **常にパブリックドメイン**
- 匿名作品（作者不明）: 公表後70年

### UMAビジネスに関する著作権の原則

#### ✅ 安全なケース

**1. パブリックドメインの伝説**
```
例: ビッグフット、ネッシー、河童、ツチノコ

理由: これらは民間伝承であり、特定の作者がいない
→ 誰でも自由に描ける
→ ただし「既存のキャラクターデザイン」を模倣してはいけない
```

**2. 完全オリジナルのデザイン**
```
プロセス:
1. 複数の資料を参考にする（1つだけはNG）
2. 自分なりの解釈を加える
3. 独自のデザイン要素を追加
4. 誰かの作品を想起させないか確認

例: "河童"を描く場合
❌ 映画「夏の庭」の河童そのまま
✅ 伝統的な河童の特徴（くちばし、甲羅、皿）を自分なりにデザイン
```

**3. 「インスピレーション」レベルの参考**
```
OK: 複数の浮世絵を参考に、自分のスタイルで描く
NG: 特定の浮世絵をトレース・模写する
```

#### ❌ 危険なケース

**1. 既存作品のキャラクターを模倣**
```
例:
❌ ハリー・ポッターのヒッポグリフ
❌ ジュラシック・パークのティラノサウルス（映画版のデザイン）
❌ ポケモンのニョロモ（河童モチーフだが独自キャラ）
❌ モンハンのイャンクック
❌ 鬼滅の刃の鬼
```

**2. 有名アーティストのスタイルを完全コピー**
```
例:
❌ 水木しげるの妖怪画をそっくり真似る
❌ ジョン・コンスタブルのビッグフット画を模倣
✅ "水木しげる風"としつつ、独自要素を加える
```

**3. トレース・盗用**
```
絶対NG:
- 既存のイラストをトレース
- AI生成画像をそのまま使用（商用利用は多くがNG）
- 他人の写真を無断使用
```

## ™️ 商標権の基礎知識

### 商標権とは

**定義**: 商品・サービスの識別標識（名前、ロゴ等）を保護する権利

**著作権との違い**:
| 項目 | 著作権 | 商標権 |
|------|--------|--------|
| 保護対象 | 創作的表現 | 商品/サービス名 |
| 自動発生 | ✅ はい | ❌ いいえ（登録必要） |
| 存続期間 | 死後70年 | 10年（更新可能） |
| 侵害例 | イラストの盗用 | 同じ商品名の使用 |

### 商標権の落とし穴

**重要**: 一般的な生物名でも、特定の区分で商標登録されている可能性がある！

**実例**:

#### ケース1: "ビッグフット"
```
検索結果（J-PlatPat）:
- 商標登録番号: 第5123456号（例）
- 権利者: A社
- 区分: 第25類（衣服、履物）
- 登録日: 2008年

結果:
✅ ステッカー（第16類）としては使用OK
❌ Tシャツ（第25類）としては使用NG
```

#### ケース2: "モスマン"
```
検索結果（J-PlatPat）:
- 商標登録: なし（日本国内）

結果:
✅ すべての区分で使用OK
```

### 主要区分（UMAビジネス関連）

| 区分 | 商品・サービス | 該当製品 |
|------|---------------|---------|
| **第16類** | 紙、文房具 | ステッカー、ポスター |
| **第25類** | 衣服、履物 | Tシャツ、パーカー |
| **第21類** | 台所用品、容器 | マグカップ、タンブラー |
| **第14類** | 貴金属、宝飾品 | エナメルピン、キーホルダー |
| **第18類** | かばん類 | トートバッグ、リュック |

**重要**: 同じ名称でも、区分が異なれば商標権の侵害にならない可能性が高い

### ⚠️ グレーゾーン（要注意）

#### 1. 近年のUMA（目撃情報が新しい）

**例**:
- スレンダーマン（2009年のクリエイティブライティング発祥）
- SCP財団の生物（クリエイティブ・コモンズだが条件あり）

**リスク**:
- 創作者が特定できる場合、著作権が発生している可能性
- 使用前に元の投稿・記事を確認

**対策**:
- 創作者のライセンスを確認
- 商用利用が許可されているか確認
- クリエジット表記が必要か確認

#### 2. 地域の伝説（地元団体が権利主張する可能性）

**例**:
- モスマン → Point Pleasant市が観光資源として管理
- ネッシー → スコットランド観光局が商標管理

**リスク**:
- 低（一般的な描写なら問題ない）
- ただし地域の公式マスコット・ロゴに似せるのはNG

**対策**:
- 公式ロゴ・マスコットと明確に区別
- "Inspired by [地域名] legend"と明記

---

### ❌ 危険な範囲（リスク: 高）

#### 1. 既存の映画・ゲーム・アニメのキャラクター

**絶対NG**:
- **映画**: ハリー・ポッター、ジュラシック・パーク、パシフィック・リム、ゴジラ（東宝版）
- **ゲーム**: ポケモン、モンスターハンター、ファイナルファンタジー
- **アニメ**: 鬼滅の刃、進撃の巨人、ドラゴンボール
- **西洋**: ドラゴンズ（D&D）、ウォーハンマー

**なぜダメか**:
- 強力な著作権保護
- 企業が積極的に権利行使
- 訴訟リスクが極めて高い

**よくある誤解**:
```
❌ 「少し変えればOK」→ NG。二次創作として違法
❌ 「ファンアートだからOK」→ 商用利用は別問題
❌ 「名前を変えればOK」→ デザインが似ていたらアウト
```

#### 2. 有名アーティストの作品スタイルを完全模倣

**例**:
- 水木しげるの妖怪画をトレース
- 特定のコンセプトアーティストのスタイルを丸ごと模倣

**リスク**:
- スタイル自体は保護されないが、「似すぎ」は訴訟リスク
- "翻案権侵害"として認定される可能性

#### 3. 商標登録済みの名称（同一区分）

**例**:
- 第16類で"○○○"が登録済み
- あなたがステッカー（第16類）で同じ名前を使用
- → 商標権侵害で販売停止・損害賠償請求

## 🔎 商標検索の実践方法

### 日本国内: J-PlatPat

**URL**: https://www.j-platpat.inpit.go.jp/

#### ステップ1: 簡易検索

1. J-PlatPatにアクセス
2. 「商標」タブを選択
3. 「商標を探す」→「称呼（単純文字列検索）」
4. 使用したい名称を入力（例: "ビッグフット"）
5. 「検索」をクリック

#### ステップ2: 結果の確認

**チェック項目**:
- ✅ **区分番号**: 第16類（ステッカー）が含まれているか
- ✅ **権利者**: 誰が登録しているか
- ✅ **商品・役務**: 具体的にどの商品が指定されているか
- ✅ **登録状態**: 「登録」なのか「出願中」なのか

#### ステップ3: 詳細確認

```
例: "ビッグフット"の検索結果

商標登録番号: 第1234567号
権利者: A商事株式会社
出願日: 2015年1月10日
登録日: 2016年3月5日
区分: 第25類
指定商品: 被服、履物、運動用特殊衣服

→ 結論: ステッカー（第16類）なら使用OK
```

#### ステップ4: 類似商標の確認

**注意**: 完全一致だけでなく、類似商標も確認！

**例**:
- "ビッグフット" → "BIG FOOT"、"ビッグフッド"なども検索
- "モスマン" → "MOTHMAN"、"モスメン"なども検索

### 海外: USPTO（アメリカ）

**URL**: https://www.uspto.gov/trademarks/search

#### 簡易検索手順

1. USPTOにアクセス
2. 「Search Trademarks」をクリック
3. 「Basic Word Mark Search (New User)」を選択
4. 検索ボックスに名称を入力
5. 「Search Term」欄に入力して検索

#### 結果の見方

```
例: "Bigfoot"の検索結果

Serial Number: 87654321
Owner: XYZ LLC
Filing Date: Jan 10, 2015
Registration Date: Mar 5, 2016
Int'l Class: 025 (Clothing)
Status: LIVE

→ 結論: ステッカーなら使用OK（Class 016ではない）
```

### 実践ワークフロー

```
📋 商標検索チェックリスト

□ J-PlatPatで日本語名を検索
□ J-PlatPatで英語名を検索（カタカナ表記も）
□ USPTOで英語名を検索
□ 該当区分（第16類等）に登録がないことを確認
□ 類似商標（表記ゆれ）も確認
□ 出願中の商標もチェック
□ 結果をスプレッドシートに記録（日付付き）
```

## ✅ デザイン作成時のチェックリスト

### Phase 1: コンセプト段階

```
□ 使用する生物名が商標登録されていないか確認済み
□ その生物がパブリックドメイン（民間伝承・古代神話）であることを確認
□ 既存作品（映画・ゲーム・アニメ）のキャラクターではないことを確認
□ 参考資料を3つ以上準備（1つだけの模倣を避けるため）
```

### Phase 2: デザイン制作段階

```
□ 既存イラストをトレースしていない
□ 特定のアーティストのスタイルを完全模倣していない
□ AI生成画像を無断使用していない（商用利用NGのものが多い）
□ 自分の手で（またはデザインツールで）1から作成した
□ 誰かの作品を見たときに「似ている」と思われないレベルか
```

### Phase 3: 商品化前の最終確認

```
□ デザインが完成してから再度商標検索（念のため）
□ 商品タイトル・説明文に商標侵害の可能性がないか確認
□ 使用している生物名が一般名詞であることを確認
□ 「公式」「オフィシャル」などの誤解を招く表現を使っていない
□ ライセンス表示が不要であることを確認
□ 第三者の著作権を侵害していないことを再確認
```

### Phase 4: 出品後の継続的モニタリング

```
□ 商標登録状況を半年に1回チェック（新規登録の可能性）
□ 同じ生物を扱う他ショップのクレーム状況を確認
□ Etsy/Printifyからの通知を見逃さない
□ レビューで「既視感がある」などのコメントがないかチェック
```

## 🛡️ 万が一の対処法

### シナリオ1: 商標権侵害の警告を受けた

#### ステップ1: 冷静に状況を確認

```
受け取った通知の内容:
□ 送信者は誰か（Etsy? 権利者? 弁護士?）
□ どの商品が問題とされているか
□ どの権利（商標? 著作権?）の侵害とされているか
□ 要求内容（販売停止? 損害賠償?）
```

#### ステップ2: 権利の有効性を確認

```
確認事項:
1. J-PlatPatで商標登録を検索
   - 登録番号が実在するか
   - 登録区分が該当するか
   - 権利が有効か（更新されているか）

2. 指定商品・役務を確認
   - あなたの商品が指定商品に含まれるか
   - 類似範囲に含まれるか

3. 権利者の確認
   - 通知の送信者が真の権利者か
   - 詐欺の可能性はないか
```

#### ステップ3: 対応策の決定

**ケースA: 明らかに侵害している場合**
```
即座に:
1. 該当商品を販売停止（Etsyから削除）
2. Printify連携も停止
3. 権利者に謝罪と対応完了を報告
4. 損害賠償請求があれば、弁護士に相談

→ 迅速な対応で損害を最小化
```

**ケースB: 侵害していない（権利主張が不当）**
```
慎重に:
1. J-PlatPatの検索結果を保存
2. 自分のデザインが独自である証拠を準備
   - 制作プロセスのスクリーンショット
   - 参考資料のリスト
   - タイムスタンプ付きファイル

3. Etsyまたは権利者に反論
   - 丁寧な言葉で
   - 証拠を添付
   - 弁護士のレビューを推奨

4. 必要に応じて弁護士に相談
```

**ケースC: グレーゾーン**
```
リスク回避:
1. 一旦商品を削除
2. 権利者と交渉
   - ライセンス取得の可能性
   - 和解の可能性

3. 弁護士に相談してから再出品を判断
```

### シナリオ2: 著作権侵害の警告を受けた

#### ステップ1: 侵害の根拠を確認

```
質問:
1. どの作品の著作権を侵害しているとされているか
2. 権利者は誰か
3. あなたのデザインと権利者の作品の類似点は何か
```

#### ステップ2: 自分のデザインの独自性を証明

```
証拠:
□ 制作過程のスクリーンショット（レイヤー、編集履歴）
□ 参考にした資料のリスト（複数の資料から独自に作成）
□ タイムスタンプ付きファイル（作成日時）
□ ラフスケッチ、下書き
□ 他の作品との相違点リスト
```

#### ステップ3: 反論または和解

**反論する場合**:
```
準備:
1. 弁護士（弁理士）に相談
2. 証拠書類を整理
3. Etsyにカウンター通知（Counter-Notice）を提出

注意:
- 偽証罪のリスクあり（確信がない場合はNG）
- 法的手続きに発展する可能性
```

**和解する場合**:
```
選択肢:
1. 商品削除で合意
2. デザイン変更で合意
3. ライセンス料支払いで合意（今後も販売したい場合）

→ 弁護士を通じて交渉推奨
```

### シナリオ3: Etsyアカウント停止

#### ステップ1: 停止理由の確認

```
Etsyからのメール:
□ 複数の商標/著作権侵害通知
□ 偽造品の販売
□ 規約違反

→ 具体的な商品・理由を確認
```

#### ステップ2: アカウント復活の申請

```
Etsyへの連絡:
1. すべての問題商品を削除済みであることを証明
2. 今後の再発防止策を提示
3. 丁寧な謝罪と改善の意志を示す

復活の可能性:
- 初回違反: 高い
- 複数回違反: 低い
- 悪質な場合: ほぼゼロ
```

#### ステップ3: 代替プラットフォームへの移行

```
選択肢:
1. 新しいEtsyアカウント（規約上は可能な場合もある）
2. 自社サイト（Shopify、BASE等）
3. 他のマーケットプレイス（Amazon Handmade、Redbubble等）

→ ただし根本原因（権利侵害）を解決しないと、どこでも同じ問題が起きる
```

## 💼 弁護士・弁理士への相談タイミング

### 相談すべき状況

```
緊急度: 高
□ 商標/著作権侵害で訴訟を起こされた
□ 損害賠償請求を受けた
□ 警告書が弁護士名義で届いた
□ 仮処分命令を受けた

緊急度: 中
□ 商標登録を自分で行いたい（ブランド保護）
□ グレーゾーンのデザインについて事前確認したい
□ ライセンス契約を結びたい

緊急度: 低
□ 一般的な法律相談（予防的）
□ 規約の作成・レビュー
```

### 費用の目安

| サービス内容 | 費用 |
|------------|------|
| **初回相談（30-60分）** | 無料～1万円 |
| **商標検索・調査** | 3～10万円 |
| **商標登録出願** | 10～20万円（登録料別） |
| **警告書への対応** | 10～30万円 |
| **訴訟対応** | 50万円～ |

### おすすめの相談先

**無料相談**:
- 日本弁理士会「知財相談」: https://www.jpaa.or.jp/
- 各地の弁護士会「法律相談」
- INPIT「知財総合支援窓口」: https://www.inpit.go.jp/

**オンライン弁護士サービス**:
- ココナラ法律相談（5,000円～）
- 弁護士ドットコム（相談無料、依頼は有料）
